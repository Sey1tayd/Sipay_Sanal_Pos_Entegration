🎯 Sipay 3D Secure - Otomatik "Yes" Bookmarklet

Bu bookmarklet, Sipay'in 3D Secure simülasyon sayfasında otomatik olarak "Yes" butonuna basar.

📋 Kullanım Talimatları:

1 Aşağıdaki bookmarklet'i tarayıcınızın bookmark bar'ına sürükleyin
2 Sipay'in 3D Secure simülasyon sayfasına gidin
3 Bookmarklet'e tıklayın - otomatik olarak "Yes" butonuna basacak

🔗 Bookmarklet:

🎯 Sipay Auto Yes

💻 Manuel Kullanım (Console):

Sipay sayfasında F12 > Console'da aşağıdaki kodu çalıştırabilirsiniz:

// Sipay 3D Secure simülasyon sayfasında otomatik "Yes" butonuna bas (function() { console.log('🎯 Sipay 3D Secure simülasyon sayfasında otomatik "Yes" butonu aranıyor...'); function findAndClickYes() { const selectors = [ 'button[name="result"][value="y"]', '#yes', '.button[value="y"]', 'input[name="result"][value="y"]', 'input[value="y"]' ]; for (let selector of selectors) { const button = document.querySelector(selector); if (button) { console.log('✅ "Yes" butonu bulundu:', button); console.log('🔄 1 saniye sonra otomatik tıklanacak...'); setTimeout(() => { button.click(); console.log('✅ "Yes" butonuna tıklandı!'); }, 1000); return true; } } // Text içeriğine göre ara const allButtons = document.querySelectorAll('button, input[type="submit"], input[type="button"]'); for (let button of allButtons) { if (button.textContent.toLowerCase().includes('yes') || button.value === 'y' || button.value === 'yes') { console.log('✅ "Yes" butonu bulundu (text içeriği ile):', button); console.log('🔄 1 saniye sonra otomatik tıklanacak...'); setTimeout(() => { button.click(); console.log('✅ "Yes" butonuna tıklandı!'); }, 1000); return true; } } return false; } // İlk deneme if (!findAndClickYes()) { console.log('⚠️ "Yes" butonu bulunamadı, 2 saniye sonra tekrar denenecek...'); setTimeout(() => { if (!findAndClickYes()) { console.log('❌ "Yes" butonu bulunamadı. Sayfayı yenileyip tekrar deneyin.'); } }, 2000); } // DOM değişikliklerini izle const observer = new MutationObserver(function(mutations) { mutations.forEach(function(mutation) { if (mutation.type === 'childList') { findAndClickYes(); } }); }); observer.observe(document.body, { childList: true, subtree: true }); console.log('👀 DOM değişiklikleri izleniyor...'); })();

🔧 Nasıl Çalışır:

  • 🎯 Sipay'in 3D Secure simülasyon sayfasında "Yes" butonunu arar
  • 🔍 Farklı selector'ları dener (id, class, name, value)
  • 📝 Text içeriğine göre de arama yapar
  • ⏱️ 1 saniye bekleyip otomatik tıklar
  • 👀 DOM değişikliklerini izler (dinamik içerik için)
  • 🔄 Bulamazsa 2 saniye sonra tekrar dener
